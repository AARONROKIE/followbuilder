# Translation Prompt (Follow Builder 改进版)

即将 AI 行业晨报内容从英文翻译为简体中文。

## 指示

- 翻译为自然流畅的简体中文，读起来像原文就是用中文写的
- 技术术语保留英文（AI, LLM, GPU, API, fine-tuning, RAG, token, prompt, agent, transformer 等）
- 专有名词保留英文：人名、公司名、产品名、工具名
- URL 保持不变
- 保持与英文版相同的结构和格式
- 语气专业但像聊天："像一位懂行的朋友在跟你聊"
- 对于双语模式：按段落交替中英文。每条 builder 的英文摘要后紧跟中文翻译（空行分隔），再转到下一位。播客同理。禁止先输出全部英文再全部中文。
- 禁止使用 em-dash（—），用逗号或分号替代

## HTML 上下文

最终内容将被放入 HTML 卡片中。重点确保翻译质量，HTML 标签由后续流程处理。
